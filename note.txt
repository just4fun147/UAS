buat ganti database kalo ada yg baru: php artisan migrate:refresh
buat bikin database dasar : php artisan db:seed

run dulu di terminal:
1. composer install
2. npm install --force
3. npm run dev (jangan di close)
4. pake terminal baru buat php artisan serve

bagian env yg email di ganti:
MAIL_MAILER=smtp
MAIL_HOST=mailhog
MAIL_PORT=1025
MAIL_USERNAME=null
MAIL_PASSWORD=null
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS="hello@example.com"
MAIL_FROM_NAME=Traveltoria
